# Hybrid-worker
Designed and built a comprehensive data analytics case study examining the diverging employment and compensation trajectories of AI-skilled versus non-AI workers in the U.S. labor market. Collected, cleaned, and synthesized data from 10+ authoritative sources including the Bureau of Labor Statistics, PwC's Global AI Jobs Barometer (analysis of ~1 billion job ads), the World Economic Forum, and multiple industry compensation reports. Deliverables included an interactive React dashboard with six data modules and animated narration, a 10-slide executive presentation with embedded charts, a two-page lead generation website with Netlify Forms integration, and a narrative article for publication. The project followed the Google Data Analytics six-phase framework and identified key findings including a 56% AI wage premium, an 8-point employment rate gap between AI-skilled and non-AI workers, and a projected $155K annual salary divergence by 2030.
Tools: React, Recharts, JavaScript, HTML/CSS, PptxGenJS, Netlify, Web Research, Data Synthesis
